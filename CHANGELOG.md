## v0.1.0 - Initial release

English
- Initial release of `pico-piano` — a lightweight piano sound library using the Web Audio API and a React Hook.
- Features:
  - `usePiano` React Hook providing simple APIs to play piano notes.
  - TypeScript support with emitted `.d.ts` type declarations.
  - Basic piano synthesis implemented with the Web Audio API.
  - Included example demos (Vite/Next) and documentation site.
- Packaging and build:
  - Built with TypeScript and prepared `dist/` exports for ESM consumers.
  - Package validated with `npm pack` and local install tests.
